Compilation instructions:
Please ensure included makefile is in the current directory.
Type 'make' without the single quotes to compile.

Manually compile by typing 'gcc ftserver.c -o ftserver' without the single quotes.

Executing instructions:
Ftserver <Server's port number>

python ftclient.py <Hostname> <Port number> <User Selection> [File Name] <Port number (Data transfer)>
Items in < >'s are mandatory.
Items in [ ]'s are optional. 

Thank you.
